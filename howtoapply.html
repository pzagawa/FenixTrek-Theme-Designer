HOW TO APPLY THEME CODE TO WATCHFACE

  1. copy Theme Code from this page and paste it into some note/document/clipboard, where it can be accessed on the phone, for example apps like Notes, Keep or GDocs
  2. copy Theme Code on the phone from the document to the clipboard
  3. open the Connect IQ Store app on the phone
  4. select "My Device" / "My Watch Faces" / "FenixTrek"
  5. select "Settings". Notice "Save" button on the page
  6. select one of the User Theme text fields from User Themes group, and paste Theme Code into it
  7. tap "Save" button
  8. the Theme Code will be transferred to a specific menu item in watchface User Theme menu
  9. enter User Theme menu on the watchface customization menu
  10. select user theme menu item with correct theme code copied in the first step from the Designer
